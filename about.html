About This Project

This project is a lightweight cybersecurity-focused workflow system for managing secure corporate laptop donations.

It supports IT departments and CSR managers by importing donation laptop inventory, recording sanitization evidence, calculating risk, approving laptops for donation, completing final handover details, and exporting records for future reference.
